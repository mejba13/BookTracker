
# BookTracker

BookTracker is a Laravel application designed to help beginners learn the basic actions and concepts in Laravel. This project serves as an introductory guide, covering essential features and functionalities, making it an excellent starting point for new developers.




## Documentation


Create Book: 
Add new books to the library with details such as title, author, genre, and publication year.

Read Books: 
Display a list of all books in the library with their details.

Update Book: 
Edit the details of existing books.

Delete Book: 
Remove books from the library.

This project is ideal for beginners looking to understand the basics of CRUD operations in Laravel 11.
